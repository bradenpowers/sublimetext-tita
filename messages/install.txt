Thank you for installing Tita -- a Titanium Mobile / Alloy Plugin


The plugin requires Titanium CLI + Titanium Alloy.

Readme: 
https://github.com/tsteur/sublimetext-tita

Features:
* Run iPhone / iPad Simulator
* Run Android Emulator / Device
* Compile + Build MobileWeb (you need to run at least once via Titanium Studio)
* Clean Build Directories
* Snippets
* Auto-Completion
